  • Работа с ответами на правки

Ответы на правки

Статусы ответа на правку:

  • accepted
  • not_accepted

Создание

Создавать может только утвержденный исполнитель заказа

[POST] http://localhost:8889/api_v2/order_result/

Json Example

{
  "comment": "хз как",
  "order_correction": 1,
  "price": 10000,
  "description": "важная правка"
}

Получение результатов по правке

[GET] /api_v2/order_result_last/{order_correction_pk}

Json Example

{
  "id": 16,
  "files_in_order_result": [],
  "confirmation": "not_accepted",
  "comment": "хз как",
  "description": "важная правка",
  "price": 10000,
  "order_correction": 1,
  "order": 6
}

Утверждение результатов правки заказчиком

Только смена статуса результата. Может использовать только заказчик.

[PATCH] /api_v2/order_correction_for_customer/{id}/

Json Example

{
  "confirmation": "accepted"
}
